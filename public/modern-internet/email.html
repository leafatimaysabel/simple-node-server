An Overview of the Internet We Know Today

Electronic Mail

Electronic Mail (E-mail) gives us the ability to contact any person in the world in a matter of seconds. It has become a popular and convenient communication tool. The E-mail system consist of computer servers that process and store messages on behalf of users who connect to the email infrastructure via and email client or web interface. When you send an e-mail, the message is transferred from his/her computer to the server linked with the recipient's address.


This uses the SMTP (Simple Mail Transfer Protocol). In the SMTP server, a Mail Transfer Agent (MTA) looks at the recipient address and looks up the domain part of the address to determine its destination. After questioning a Domain Name System (DNS) server for the name of the Mail eXchanger (MX) for the recipient's domain name, the SMTP server will send the message to that server via the SMTP. The receiving server will store the message and make it available to the recipient, who can access it via web, POP or IMAP.
